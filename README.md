This is a Firefox plugin which fixes the DEVPROM with well formatted URL.

The following Devprom URL
1. https://devprom.e-kama.com/pm/IT_XXX/IT_IVI.Group - 1221916
2. https://devprom.e-kama.com/pm/HMXconcept/XXXSpec-LVL3-1125932?baseline=35307

are formated to well URL based links
1. [[https://devprom.e-kama.com/pm/IT_XXX/IT_IVI.Group%20-%201221916 Cabin Air Recirculation ON OFF]]
2. [[https://devprom.e-kama.com/pm/HMXconcept/XXXSpec-LVL3-1125932?baseline=35307 Manage Cabin Air Recirculation AUTO]] 

The main features are:
1. Supports both TXT & HTML formats
2. Supports GUI for template format
2.1. User defined format
2.2. Jira format
2.3. MD format
3. Supports shortcuts
3.1. Win -`Ctrl+Alt+D`
3.2. Linux/Mac -`Command+Alt+D`
